Perguntas frequentes

Preciso ter internet para acessar o aplicativo?

Sim. Mas se o app for baixado a partir da loja oficial de sua plataforma, algumas funcionalidades funcionarão sem conexão.

Preciso do CNPJ para começar?

Não, o CNPJ não é obrigatório para criar sua conta e utilizar o serviço. Caso queira, você pode inserir depois editando seu perfil.

Como funciona a análise do negócio?

Está é uma funcionalidade gratuita. Ao responder algumas perguntas referentes a cada setor da sua empresa, uma pontuação será gerada para cada resposta e isso completará um gráfico de mapeamento. Ao terminar o questionário você terá uma representação visual da situação do seu negócio área por área.

Eu preciso pagar para usar o aplicativo?

Não há custo para você usar o aplicativo em sua versão básica (plano gratuito). Caso queira ou necessite das funcionalidades premium você deve assinar um de nossos planos.

Posso ter acesso a base de conhecimento gratuitamente?

Sim, o acesso a base de conhecimento faz parte do plano básico e você poderá aproveitar todos os nossos conteúdos gratuitamente.
